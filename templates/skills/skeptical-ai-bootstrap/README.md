# Skeptical AI Bootstrap Templates

This directory contains supporting templates for the project-scope bootstrap skill:

- `ARCHITECTURE-INVENTORY.md`
- `DECISION-GUARDS.md`
- `REPORT-TEMPLATE.md`
- `NEW-SKILL-TEMPLATE.md`
- ecosystem and installation examples

The executable skill lives in `.agents/skills/skeptical-ai-bootstrap/SKILL.md`.
This folder is intentionally not a standalone skill package, so it does not contain
its own `SKILL.md` or `CHECKLIST.md`.
